   "Cmd + / " (for mac)"
2. What are the two kinds of loops available in Python?
A. for loop(definite)
   while loop (indefinite)
3. What is a "sentinel" (when dealing with loops)?
A. A special value used to signal the end of input or processing in a loop.
   It's  help user control the loop's termination based on a specific condition
4. What is wrong with this variable name: nothingWrong
A. It's not follow the python convention, it should be "nothing_wrong".
5. What are augmented assignment operators?
A. Augmented assignment operators are shorthand operators in Python
   that perform an operation and assign the result back to the variable.
