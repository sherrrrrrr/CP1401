4. What are arguments and parameters? What is the difference between them?
A. 1. Actual parameters are actual values or objects passed to a function or method
   when it is called. Simply put, actual parameters are the input values passed to the
   function. When you call a function, you need to provide the actual parameters so
   that the function can use them to perform calculations or operations.
   2. Parameters are variables in a function or method definition. They are used to
   receive the values of actual parameters. Formal parameters are input parameters
   to a function or method that define the input expected by the function or method.
   When you define a function, you need to specify some parameters to accept input values.
   These parameters can be used inside the function to perform calculations or operations.
   3. The difference is that arguments are actual values passed to the function, while parameters
   are variables in the function definition. When a function is called, the actual parameters
   are assigned to the parameters, and the parameters can be used for calculations or operations
   inside the function.
